# vtex-checkout-kit

Frontend-only VTEX Checkout boilerplate based on the requested checkout project.

## What is included

- `manifest.json` configured only with frontend builders:
  - `react`
  - `messages`
  - `checkout-ui-custom`
- `react/` starter component
- `messages/` starter locale files
- `checkout-ui-custom/` starter JS and CSS assets for Checkout v5 and v6

## What is intentionally excluded

- No `node` builder
- No `node/` folder
